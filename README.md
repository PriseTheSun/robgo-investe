# RobGO Investe

**Inteligência em Investimentos**

Analista de investimentos inteligente para todos os níveis: Iniciante, Intermediário e Avançado.

## Sobre o Projeto

O RobGO Investe é uma aplicação web desenvolvida com Next.js e React que oferece análises de investimentos impulsionadas por IA (através do Google Gemini). O projeto visa fornecer insights e estratégias de investimento adequadas para diferentes perfis de investidores.

## Tecnologias Utilizadas

- **Next.js 15** - Framework React para produção
- **React 19** - Biblioteca para construção de interfaces
- **Tailwind CSS 4** - Framework de utilitários CSS para estilização
- **Framer Motion** - Biblioteca de animações
- **Gemini AI SDK (`@google/genai`)** - Integração com a IA do Google Gemini
- **Lucide React** - Conjunto de ícones
- **Recharts** - Biblioteca de gráficos
- **TypeScript** - Tipagem estática para JavaScript

## Como Configurar

1. Clone o repositório ou faça o download dos arquivos.
2. Copie o arquivo `.env.example` para `.env` e configure suas variáveis de ambiente:
   - `GEMINI_API_KEY`: Sua chave de API do Google Gemini.
   - `APP_URL`: A URL onde o aplicativo será executado (ex: `http://localhost:3000`).

## Como Executar

Instale as dependências:
```bash
npm install
```

Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o aplicativo em execução.
